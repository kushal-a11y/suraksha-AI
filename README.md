# 🛡️ SurakshaAI

### AI-Powered Cyber Fraud & Government Assistance for Bharat 🇮🇳

---

## 🚀 Overview

**SurakshaAI** is an intelligent assistant built using the **ASI-1 API** that helps users:

* Detect and understand online scams
* Receive actionable cybersecurity advice
* Discover relevant government schemes based on their situation

This project focuses on **AI for Bharat**, making digital safety and awareness accessible to everyone through simple, structured, and intelligent analysis.

---

## 🎯 Objectives

* 🛡️ Protect users from cyber fraud (OTP scams, job scams, impersonation, phishing)
* 🧠 Use ASI-1 as a **reasoning engine**, not just a chatbot
* 🇮🇳 Provide India-specific guidance and awareness
* 📢 Educate users with clear explanations and next steps
* 🏛️ Help users discover **relevant government schemes**

---

## 💡 Problem Statement

Millions of users in India fall victim to:

* Fake job offers
* OTP scams
* WhatsApp impersonation
* Phishing links

Existing systems:

* Are either too technical
* Or fail to provide **clear, actionable guidance**
* Do not combine **fraud detection + user education + government help**

---

## 🧠 Our Solution

SurakshaAI uses **structured prompting with ASI-1** to:

1. Analyze suspicious messages
2. Classify scam types
3. Assess risk levels
4. Explain threats in simple language
5. Provide actionable safety steps

Additionally, it can:

* Suggest **government schemes** based on user needs
* Act as a **digital safety companion**

---

## 🧪 How It Works

1. User enters a suspicious message
2. The input is sent to ASI-1 API
3. AI analyzes using structured reasoning:

   * Scam Detection
   * Scam Type
   * Risk Level
   * Explanation
   * Recommended Action
4. Results are displayed in a clear format

## 🛠️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/your-username/suraksha-ai.git
cd suraksha-ai
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Add API Key

Create a `.env` file:

```
ASI_API_KEY=your_api_key_here
```

### 4. Run Server

```bash
node server.js
```

### 5. Open Frontend

Open `index.html` in browser
(or run your frontend setup)
